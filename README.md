# Intersect

Intersect is a Java/libGDX analytical geometry project created as a personal software development project.

The goal of Intersect is to build an interactive workspace for experimenting with geometric relationships, including points, lines, grids, intersections, and visual problem solving.

## Current Features

- Interactive libGDX desktop application
- Coordinate-style visual workspace
- Movable points/dots
- Grid-based visual layout
- Line drawing and visual geometry experimentation

## Project Structure

- `core`: Main application logic shared by the project
- `lwjgl3`: Desktop launcher using LWJGL3
- `assets`: Project assets used by the application
